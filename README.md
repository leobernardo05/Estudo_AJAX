# 🌐 Estudos de AJAX com JavaScript

Este repositório reúne exemplos e exercícios práticos desenvolvidos durante meus estudos sobre **AJAX (Asynchronous JavaScript and XML)**, explorando como realizar requisições assíncronas para carregar dados e atualizar páginas sem a necessidade de recarregamento completo.

O projeto apresenta diferentes formas de consumir informações utilizando **XMLHttpRequest**, trabalhando com arquivos **HTML**, **JSON** e **XML**, além da manipulação dinâmica do DOM.

## 🎯 Objetivos de Aprendizagem

- Compreender o funcionamento do AJAX.
- Realizar requisições assíncronas com `XMLHttpRequest`.
- Manipular os estados (`readyState`) e códigos de resposta (`status`).
- Consumir dados em formato JSON.
- Consumir e interpretar documentos XML.
- Converter XML para JSON utilizando JavaScript.
- Atualizar elementos da página dinamicamente.
- Implementar indicadores de carregamento (*loading*).
- Compreender a diferença entre requisições síncronas e assíncronas.

## 📚 Conteúdos Desenvolvidos

### 📄 Requisições Assíncronas

- Carregamento dinâmico de páginas HTML
- Atualização parcial do conteúdo da página
- Indicador de carregamento (Loading)
- Tratamento de erros (404)

### 🎬 Catálogo de Filmes

Aplicação que realiza requisições AJAX para carregar um catálogo de filmes a partir de:

- Arquivo JSON
- Arquivo XML

Durante o processamento são utilizados:

- Criação dinâmica de elementos HTML
- Manipulação do DOM
- Conversão de XML para JSON
- Percurso de objetos e arrays em JavaScript

## 🚀 Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/leobernardo05/Estudo_AJAX.git
```

2. Abra o projeto em um servidor local (XAMPP, Laragon, WAMP ou VS Code Live Server).

3. Execute os arquivos HTML pelo navegador.

> **Importante:** Algumas funcionalidades AJAX dependem de um servidor HTTP. Abrir os arquivos diretamente (`file://`) pode impedir as requisições devido às políticas de segurança do navegador.
